## Getting Started

### Prerequisites

- Rust
- Node.js (18+)
- System dependencies for Tauri (webkit2gtk, etc., you can find more details in the [setup.sh](./setup.sh) script)

### Development

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run in development mode:
   ```bash
   npm run tauri dev
   ```

## Building

To build for production:

```bash
npm run tauri build
```
